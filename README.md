# Job Application Challenge

Aplicación desarrollada con React y TypeScript que permite:

- Obtener la información de un candidato por email.
- Listar trabajos disponibles.
- Postularse enviando la URL de un repositorio de GitHub.

## Instalación

```bash
npm install
npm run dev
```

## Notas

- Se manejan estados de carga y error.
- Se valida la URL antes de enviar la postulación.
- Se evita el doble envío desde la interfaz.
